# 🛠️ カオス・レイド・レース｜GM・主催者向け運営手引き

---

## ■ 概要

| 項目 | 内容 |
|---|---|
| 参加人数 | 12名（3チーム × 4人） |
| 総プレイ時間 | 120分 |
| サーバー構成 | Fabric サーバーサイドMOD（参加者はバニラ可） |
| 必須MOD | Simple Voice Chat |
| ワールド | オーバーワールド（500×500固定）＋ エンド（中央島のみ） |

---

## ■ 事前準備チェックリスト

### サーバー・ワールド

- [ ] Fabricサーバーを立ち上げ、MODを導入済みか確認
- [ ] オーバーワールドのワールドボーダーを半径250に設定
- [ ] 時間を18000（真夜中）に固定・昼夜サイクルをOFF
- [ ] 天候をclearに固定・天候サイクルをOFF
- [ ] 各チームの拠点を方角固定で建設（東・西・南）
  - [ ] 拠点内にベッドを1つだけ設置
  - [ ] 拠点内にライトブロックを設置（Mobスポーン抑制・参加者が破壊できない）
  - [ ] 拠点の保護座標をconfig.ymlに登録
- [ ] エンドの中央島を整備（平坦化・遮蔽物・縁石・チーム別スタート地点）
- [ ] `/crr snapshot save` でワールドスナップショットを保存

### config.yml

- [ ] チーム名・カラーを設定
- [ ] 拠点座標・保護半径・ベッド座標を登録
- [ ] 最終フェーズのリスポーン候補座標を10〜15点登録
- [ ] Mob・ショップ価格等のパラメータを最終確認

### チーム編成

- [ ] `/crr team add <player> <team>` で全参加者をチームに割り当て
- [ ] `/crr team list` で編成を確認
- [ ] Simple Voice Chatの範囲設定を確認

---

## ■ 当日タイムライン・GM作業手順

### ゲーム開始前

```
参加者全員の接続確認
↓
/crr team list でチーム編成を最終確認
↓
/crr ready <player> を全員分実行
↓
/crr start でゲーム開始
```

### ゲーム中の定期確認

| タイミング | GM作業 |
|---|---|
| 5:00 | PvP解禁・第1カオスルール発動を確認 |
| 20:00 | 第1中ボス自動出現を確認。不具合時は `/crr boss spawn 1` |
| 40:00 | 第2中ボス・2枚同時ルール解禁を確認 |
| 60:00 | 中間スコア発表・最下位救済フラグを確認 |
| 80:00 | 第3中ボス（ウォーデン）出現を確認 |
| 100:00 | 討伐Pt1.5倍・ラストスパート宣言を確認 |
| 108:00 | エンドワープ予告を確認 |
| 110:00 | 全員エンドワープ・エンドラ召喚・バグ演出ONを確認 |
| 120:00 | ゲーム終了・最終スコア発表を確認 |

### ゲーム終了後

```
/crr reset でリセット（スナップショット復元＋スコア初期化）
↓
次回のために /crr snapshot save を再実行
```

## ■ 緊急時対応コマンド集

| 状況 | コマンド |
|---|---|
| ゲームを一時停止したい | `/crr pause` |
| ゲームを再開したい | `/crr resume` |
| 中ボスがスポーンしない | `/crr boss spawn <回>` |
| 中ボスを即時消去したい | `/crr boss kill` |
| カオスルールを飛ばしたい | `/crr rule skip` |
| 特定ルールを強制発動したい | `/crr rule <ルール名>` |
| プレイヤーが奈落に落ちた | `/crr respawn <player>` |
| スコアを手動で加減算したい | `/crr score <team> <+/-><pt>` |
| 設定を変更してすぐ反映したい | `/crr set <key> <value>` → `/crr reload` |
| 現在の状態を確認したい | `/crr status` |
| バグ演出を手動で切り替えたい | `/crr glitch <on/off>` |
| 途中で切断したプレイヤーを復帰させたい | 再接続後に自動復元（猶予2分） |
| 途中参加者を追加したい | `/crr add <player> <team>` |
